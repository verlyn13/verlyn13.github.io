# Symmetry Groups

Creative Mathematical Visualization

Creative Project · Active

## Overview

Symmetry Groups explores mathematical structure through an interactive educational interface. It belongs in the portfolio because it shows mathematical communication, interface thinking, and rapid prototyping. It also needs a methodological honesty note: this project uses claude.ai/design as the primary tool, which is different from the fuller substrate-driven pattern used in the Nash Group and host-capability work.

## Methodology Note

This project was authored in collaboration with [claude.ai/design](https://claude.ai/design) as a design tool. The mathematical content, structural decisions, and visualization concepts originated with the operator; claude.ai/design produced design and code execution against those briefs. This is disclosed here so that the visual and structural work can be evaluated honestly — as a study of what's achievable when a mathematician partners with an LLM-driven design tool, rather than as a pure solo art project.
