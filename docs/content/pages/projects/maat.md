# MAAT Framework

Historical multi-agent research experiment

Historical artifact · Evidence review pending

## What is this?

MAAT Framework is a historical record of early experimentation with distributed agents, orchestration, governance concepts, and the idea of truth and balance as system-design constraints.

## What is it for?

It preserves the development history behind later work on explicit agent authority, mediation, evidence, and machine-checkable governance without presenting the early experiment as a current product.

## How is it used?

Today it is used as a historical reference point: a record of questions and prototypes that helped shape more recent, evidence-backed evaluation and governance systems.

## Why does it matter?

Development history can show how a research program evolved, but only if exploratory ideas are kept separate from verified implementation claims. MAAT helps explain that progression without competing with the current portfolio evidence.

## Evidence and status

The accepted portfolio authority classifies MAAT as a historical development artifact rather than a lead maturity claim. A fresh source review has not yet established which earlier implementation details remain supportable for outward use.

### Claims intentionally withheld

No current-product, active-development, tension-mediation, gRPC, MCP, observability, policy-enforcement, mathematical-formalization, performance, or adoption claim is made until source-local evidence is reviewed.

## Questions preserved by the record

- How might agents coordinate when authority is distributed?
- Which orchestration and governance patterns can be stated precisely enough to verify?
- How should principles, evidence, memory, and decision-making interact?
