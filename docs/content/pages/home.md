## Selected work

A few projects show the work directly. Each project page separates what is implemented from what remains open.

- **Host Capability Substrate** Public TypeScript implementation of explicit host capabilities, authority, policy decisions, and evidence. It includes 46 Zod entity schemas, 67 generated JSON Schemas, and about 500 tests. [Examine the public implementation →](/projects/host-capability-substrate.html)
- **Budget Triage** Financial evidence workbench with exact monetary handling, provenance, tenant isolation, and human review. A public source snapshot is available; active development is private; the system is not deployed. [Review the workbench →](/projects/budget-triage.html)
- **Agentic-Coding Evaluation Lab** Public fail-closed model-evaluation method. A separate verifier recomputes a synthetic `NOT_EVALUABLE` result when required evidence is missing. It does not establish real-model performance. [Explore the method and evidence →](https://github.com/verlyn13/eval-lab-methodology)
- **Dicee** Live multiplayer browser game using Rust/WebAssembly, SvelteKit, Cloudflare Durable Objects, and WebSockets, with public source and a live deployment. [Review the delivered system →](/projects/dicee.html)
